# QartNewSurfer
Silver Surfer of Qartvelian Web Media

## Dependencies

```shell script
pip install -r requirements.txt
```

## Instruction
navigate to the QartNewSurfer toolset directory:
 ```shell script
cd qartnewsurfer
```

and run the desired surfer:
 ```shell script
scrapy crawl <surf_board>
```
